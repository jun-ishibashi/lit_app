# サービス名 — CargoLink（カゴリンク）

サービス名は **CargoLink**（カゴリンク）に決定しています。表示名は **config/application.rb** の `config.x.app_display_name` で変更できます。

---

## 選定時の候補（参考）

| 名前 | 読み・表記 | イメージ・理由 |
|------|------------|----------------|
| **CargoLink** | カゴリンク | 貨物（cargo）とつながり（link）。比較・マッチングのイメージ。短く覚えやすい。 |
| **はこぶね** | はこぶね / Hakobune | 「運ぶ船」。海外輸送を連想しやすく、日本語で親しみやすい。 |
| **輸送ナビ** | ゆそうナビ | 輸送の案内であることが一目で伝わる。検索・比較のイメージと合う。 |
| **くらべルート** | くらべルート | 「比べる」＋ルート。比較サービスであることが明確。 |
| **LogiBridge** | ロジブリッジ | 物流（logistics）と橋渡し（bridge）。業者と依頼者をつなぐイメージ。 |

---

## 選定のポイント

- **依頼側**：検索・比較・見積依頼がしやすいサービスだと伝わるか
- **業者**：掲載・リード獲得のプラットフォームだと伝わるか
- **覚えやすさ**：口頭・URL・名刺で伝えやすいか
- **ドメイン**：取得可能なドメインがあるか（別途確認）

---

## 現在の設定

- **表示名**：**CargoLink**（カゴリンク）。`config/application.rb` の `config.x.app_display_name` で指定。ヘッダー・フッター・ページタイトル・メタ説明はすべてこの設定を参照する。
- **コード・リポジトリ名**：`lit_app` のまま（変更すると影響が大きいため）。画面・ドキュメント上のサービス名のみ変更可能。
